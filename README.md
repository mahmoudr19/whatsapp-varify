# WhatsApp Gateway — WordPress Plugin 🚀

إضافة ووردبريس احترافية تعمل كبوابة تحقق من أرقام الجوال قبل التوجيه إلى واتساب أو الاتصال المباشر.

## ✨ المميزات

| الميزة | الوصف |
|---|---|
| **بوابة مزدوجة** | واتساب 🟢 + اتصال مباشر 🔵 في نفس الإضافة |
| **بوب أب احترافي** | نافذة منبثقة بتصميم عصري تظهر على أي صفحة |
| **تحقق من الرقم** | يقبل فقط أرقام الجوال السعودية (05XXXXXXXX) |
| **حفظ الأرقام** | يخزن كل الأرقام في قاعدة البيانات مع تصنيفها (واتساب / اتصال) |
| **لوحة تحكم** | داشبورد لعرض الأرقام المسجلة مع تصدير CSV |
| **تخصيص كامل** | إعدادات مرنة لرقم الوجهة و ID الزرار |

## 📦 التثبيت

1. حمّل الإضافة كملف `.zip`
2. من لوحة تحكم ووردبريس: **إضافات → أضف جديد → رفع إضافة**
3. فعّل الإضافة
4. اذهب إلى **الإعدادات → WhatsApp Gateway**

## ⚙️ الإعدادات

| الإعداد | الوصف | القيمة الافتراضية |
|---|---|---|
| رقم واتساب | الرقم اللي هيتحول عليه العميل | — |
| ID زرار واتساب | معرف الزرار اللي يفتح بوب أب واتساب | `whatsapp-btn` |
| رقم الاتصال | رقم التليفون للاتصال المباشر | — |
| ID زرار الاتصال | معرف الزرار اللي يفتح بوب أب الاتصال | `call-btn` |

## 🔧 الاستخدام

أضف أزرار في أي مكان بالموقع:

```html
<button id="whatsapp-btn">تواصل معنا عبر واتساب</button>
<button id="call-btn">اتصل بنا الآن</button>
```
